# Game_of_Life
This is a pygame version of Game of life with python. The Game of Life, also known simply as Life, is a cellular automaton devised by the British mathematician John Horton Conway in 1970. It is a zero-player game, meaning that its evolution is determined by its initial state, requiring no further input.


Once the screen is shown , draw an initial state then  press enter to set that state. While drawing the initial state ,  and press **1** for Draw Mode.
### Erase Mode:-
Press **0** 

### Draw Mode:-
Press **1** 

### Saving the state :-
Press Space

### Load the saved state:-
Run Load.py
